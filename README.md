Project Overview

This project is a basic front-end web application developed using HTML and Bootstrap. It serves as a template for building responsive and visually appealing web pages quickly.

# Installation
To run this project locally, follow these steps:

Clone the repository to your local machine:

git clone https://github.com/habibir7/fe-resepku.git

Navigate to the project directory:

cd fe-resepku

Open the index.html file in your preferred web browser.

# Usage
Once you've opened the project in your web browser, you can navigate through the different pages using the navigation menu. The pages are designed using Bootstrap's grid system and components for responsiveness and consistency.

# Features
Responsive design using Bootstrap.

Clean and minimalistic UI.

Easy-to-understand HTML structure.

Modular CSS classes for easy customization.
